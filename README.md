# typist.

A minimal, distraction-free typing speed test built for speed, focus, and precision.

Designed with a clean, modern interface and real-time feedback system to measure typing performance through WPM and accuracy.

---

## 🚀 Live Demo

If deployed, you can try it here:  
👉 https://typist-zkqj.onrender.com

---

## 🧠 Features

- Real-time WPM calculation
- Accuracy tracking (correct vs incorrect keystrokes)
- Timed modes: 15s / 30s / 60s
- Dynamic word generation system
- Cursor-based typing experience
- Instant restart + replay system
- Minimal UI focused on performance, not distractions

---

## 🛠 Tech Stack

- HTML5 / CSS3
- Vanilla JavaScript
- Python (Flask)
- Google Fonts (JetBrains Mono, Syne)
